# 新传考研作业批改 Skill

## 概述

本 Skill 用于批改新传考研学生的各类答题作业，核心目标不是把学生答案改成论文，
而是让答案在考场的有限时间和版面中最大化呈现得分点。

批改遵循**先结构后表达、先宏观后微观**的原则，重点关注审题准确性、框架完整性、
理论准确性和论证闭合度四大方面。

## 触发场景

以下情况应加载此 Skill：

- 用户上传学生作业文件（.docx / .pdf），要求批改或打分
- 用户询问「这道题怎么改」「帮我看看这份作业」
- 用户提到「批改」「评分」「赋分」「修改建议」等关键词
- 用户询问批改标准或答题模板（如「30分论述题怎么写」「名词解释怎么答」）

## 批改工作流

### 步骤 1：读取并分析作业

1. 打开学生提交的文件（.docx 或 .pdf），完整读取内容
2. 识别题目、题型和分值
3. 读取学生答案的全部文本

### 步骤 2：按八步流程逐项审查

严格按照以下顺序审查答案，每一步都要给出判断和具体标注：

1. **判断题型与分值**：先确定它是名解、简答、论述、评论、策划、编译还是研究设计
2. **审题检查**：标出题眼、作答方向、是否漏问或多问式题目有偏废
3. **结构审查**：开头、主体、结尾是否完整；点数与体量是否匹配分值
4. **小标题审查**：是否概括观点、格式统一、有区分度
5. **论证审查**：每个点的论证链是否完整（观点→解释→案例→回扣）
6. **知识准确性检查**：概念、学者、学派、理论来源是否有硬伤
7. **表达与版面检查**：语言是否精炼规范、能否一眼抓到得分点
8. **总评与赋分**：先指出核心问题，再给可执行改法，最后给出赋分参考

### 步骤 3：输出批改结果

在原始文件上添加批注，并给出总评：

**逐点批注**（紧贴在原文对应位置）：
- 用红色/醒目颜色标记问题
- 每条批注明确指出：什么问题 → 为什么是问题 → 怎么改
- 肯定优点的批注同样重要，好的地方要标注出来

**总评**（放在答案末尾或文件开头）：
- 格式：「整体来看，这篇答案的优点在于【优点】。目前最影响分数的问题是【核心问题】。后续修改建议先处理【优先级最高的】，再做局部润色。」
- 给出赋分参考（按对应题型的赋分标准逐项打分）

## 题型速查表

| 题型 | 分值范围 | 推荐点数 | 体量参考 |
|------|---------|---------|---------|
| 名词解释 | 5分 | 1-2段 | 1/3面 |
| 10分简答 | 10分 | 3-4点 | 1面 |
| 15分简答/小论述 | 15分 | 3-4点 | 1.5面 |
| 20/25分题 | 20-25分 | 4-5点 | 2-2.5面 |
| 30分论述 | 30分 | 5-6点(2层次) | 3面/1200字 |
| 40分大论述 | 40分 | 6-8点(2-3层次) | 4面 |
| 新闻评论 | 30分 | 核心论点+分论点 | 按标题/结构/论点/结尾赋分 |
| 编译/消息改写 | 30分 | 按标题/平台/结构/事实/表达赋分 |
| 报道策划/采访提纲 | 30分 | 按主题/目的/平台/内容/采访/效果赋分 |
| 研究设计 | 30-50分 | 按标题/背景/问题/概念/方法/结果赋分 |

## 批注格式规范

在原文添加批注时遵循以下原则：

- **红色批注用于问题和修改建议**：明确指出问题、原因和改法
- **蓝色/绿色批注用于肯定优点**：好的地方要标注，不能只改不改夸
- **每条批注三要素**：问题所在 → 为什么是问题 → 怎么改进
- **赋分标注**：在答案末尾给出逐项赋分，格式为「赋分：开头X分 + 主体X分 + 结尾X分 = 总分」

## 批改话术

使用自然但专业的批改语言，参考以下风格（完整模板见 `references/grading-standard.md` 第七章）：

- 「开头的名词解释要尽可能精确直接，简明扼要，一剑封喉！不要在次要位置上恋战！」
- 「小标题重要的功能是简明扼要地阐释观点，且表明观点的面向」
- 「案例和分论点一定要精准地搭配」
- 「这道题的题眼是【】，重点问的是【】。你目前更多在写【】，对【】回应不够」
- 「实务题最重要的是可执行性。看完你的策划，阅卷老师应当能想象成品长什么样」

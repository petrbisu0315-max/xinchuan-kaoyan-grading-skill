## 它能做什么

把这份 Skill 安装到 WorkBuddy 后，你只需要把作业文件（Word 或 PDF）拖进对话框，说「帮我批改」，它就会：

1. **自动识别题型和分值**，匹配对应的评分标准
2. **按八步流程逐项审查**你的答案：审题→结构→小标题→论证→知识准确性→表达→总评
3. **在原文件上加批注**：红色标注问题和扣分点，蓝色给出修改示例和优化建议
4. **生成结构化批改报告**：包含逐项赋分表、优点总结、问题分析、优先修改建议
5. **给出具体修改示范**：不是只说「这里不好」，而是告诉你「改成这样更好」

## 支持的题型

| 题型 | 典型分值 | 批改重点 |
|------|----------|----------|
| 名词解释 | 5分 | 定义准确度、提出者与来源、内涵完整性、当代意义 |
| 简答题 | 10～15分 | 审题精准度、结构完整度、知识点覆盖、结尾回扣 |
| 论述题 | 20～40分 | 框架层次、小标题质量、论据论证链、理论深度、体量匹配 |
| 新闻评论 | 30分 | 标题力度、立论角度、论证严密性、结尾升华 |
| 新闻编译 / 消息改写 | 25分 | 标题、拟报道平台、事实概述、结构、表达 |
| 报道策划 / 采访提纲 | 30分 | 报道主题、平台端口、内容规划、产品形态、采访方案 |
| 研究设计 | 30～50分 | 标题、研究背景、研究问题、理论框架、方法设计 |

## 批改流程

每次批改严格遵循以下八步流程：

```
第一步：判断题型与分值
        ↓
第二步：看审题
        → 题眼抓住了吗？作答方向对吗？有没有漏问？
        ↓
第三步：看结构
        → 开头-主体-结尾完整吗？点数与分值匹配吗？
        ↓
第四步：看小标题
        → 是否概括了核心观点？格式统一吗？
        ↓
第五步：看主体论证
        → 每个论点有解释+案例+回扣吗？论证链闭合吗？
        ↓
第六步：看知识准确性
        → 概念、学者、学派有没有张冠李戴？
        ↓
第七步：看表达与版面
        → 有没有病句、口语化、套话？卷面可读性如何？
        ↓
第八步：给总评和修改方向
        → 逐项赋分 + 优点 + 核心问题 + 优先修改建议
```

## 安装方法

### 前提条件

1. 安装 **WorkBuddy**（AI 助手，支持 Windows 和 Mac）
2. 下载本仓库的 `xinwen-kaoyan-grading.zip`（在 [Releases](https://github.com/petrbisu0315-max/xinwen-kaoyan-grading-skill/releases) 页面下载最新版本）

### 安装步骤

**Windows 用户：**

1. 解压 `xinwen-kaoyan-grading.zip`，得到文件夹 `xinwen-kaoyan-grading`
2. 将该文件夹复制到 `C:\Users\你的用户名\.workbuddy\skills\` 目录下
3. 如果 `.workbuddy\skills\` 文件夹不存在，请手动创建
4. 重启 WorkBuddy，或在工作台刷新技能列表

**Mac 用户：**

1. 解压 `xinwen-kaoyan-grading.zip`，得到文件夹 `xinwen-kaoyan-grading`
2. 将该文件夹复制到 `~/.workbuddy/skills/` 目录下
3. 如果 `skills` 文件夹不存在，请手动创建
4. 重启 WorkBuddy，或在工作台刷新技能列表

### 验证安装

安装完成后，在 WorkBuddy 对话框中输入「帮我批改作业」，如果系统正确加载了批改 Skill，就会按照本标准的流程对你的作业进行批改。

## 使用方法

### 基本用法

1. 打开 WorkBuddy
2. 将你的作业文件（`.docx` 或 `.pdf` 格式）拖入对话框
3. 输入以下任意指令：
   - `帮我批改这份作业`
   - `批改一下`
   - `请批改`
4. 等待 AI 分析完成，你将收到：
   - 一份带批注的原文件（红色=问题，蓝色=建议）
   - 一份结构化批改报告（含逐项赋分和修改建议）

### 进阶用法

| 使用场景 | 指令示例 |
|----------|----------|
| 只想知道分数 | `帮我打分就行，不用详细批注` |
| 重点看某个维度 | `帮我重点看论证和案例部分` |
| 不看表达只看内容 | `不用管语言表达，只看论点框架` |
| 对比两次批改 | `这是我修改后的版本，帮我对比一下进步了哪些` |
| 让 AI 给范文 | `针对这道题，帮我写一段高分开头做参考` |

## 批改标准概要

### 五大批改维度

| 维度 | 考察重点 | 高分特征 | 常见扣分点 |
|------|----------|----------|------------|
| 审题与立意 | 题眼拆解、作答方向 | 精准拆解题干关键词，多问式不偏废 | 只答熟悉的内容、开头背景拉太远 |
| 框架与结构 | 开头-主体-结尾完整度 | 3～5个互斥层次，体量与分值匹配 | 点数过多过浅、缺开头或结尾 |
| 理论与知识 | 概念准确性、学者脉络 | 定义精确，交代提出者与理论脉络 | 张冠李戴、定义过长遮蔽得分点 |
| 论证与案例 | 论证链闭合、案例贴合 | 观点→解释→案例→回扣完整闭合 | 只有观点无解释、案例与论点脱节 |
| 表达与卷面 | 语言规范、卷面可读 | 学术化但不晦涩，短句优先，序号规范 | 病句、口语化、字小行密 |

### 各题型赋分参考

**名词解释（5分）**
- 定义：2分 | 内涵：1.5分 | 意义/评价：1分 | 表达：0.5分

**10分简答题**
- 开头：1～1.5分 | 主体（3～4点）：7～8分 | 结尾：1～1.5分

**15分简答/小论述**
- 开头：1.5分 | 主体（3～4点）：12分 | 结尾：1.5分

**20分论述题**
- 开头：2分 | 主体（4～5点）：14～16分 | 结尾：2分

**30分论述题**
- 开头：3分 | 主体（4～6点/两层结构）：24分 | 结尾：3分

**40分大论述**
- 开头：4分 | 主体：32分 | 结尾：2分 | 结构表达：2分

## 常见问题

### Q：这个 Skill 收费吗？

**完全免费。** 这是彼得学长将两年批改经验开源的工具，目的是帮助更多新传考研同学在练题阶段获得即时反馈。

### Q：AI 批改能替代人工批改吗？

**不能完全替代。** 这个工具擅长标准化审查（审题偏没偏、结构完不完整、知识有没有硬伤），但在深度专题讲解、考场策略定制、个性化长期追踪等方面，仍然建议找有经验的学长学姐进行一对一批改。本工具定位是「日常练题的随身批改助手」。

### Q：批改结果的准确率如何？

批改标准基于 287 份真实批改稿整理而来，评分逻辑与彼得学长本人的批改风格高度一致。已通过多份实际作业的对比测试，得分误差通常在 ±1 分以内。

### Q：支持手写作业吗？

**支持。** WorkBuddy 具备图片识别能力，你可以直接把手写作答的拍照图片（`.png`、`.jpg`）或扫描 PDF 拖进去批改。不过需要提醒两点：

1. **字迹要清楚**：潦草或模糊的字迹会影响识别准确度，建议用扫描类 App（如扫描全能王）拍成清晰的 PDF 再提交。
2. **卷面本身就是评分维度之一**：考试卷面整洁度在「表达与卷面」维度中占 0.5～2 分不等，手写版正好能反映真实考场状态。

### Q：会不会把我的作业泄露出去？

不会。WorkBuddy 在本地运行，你的作业文件不会上传到任何云端服务器。本 Skill 的所有批改逻辑也是纯本地执行。

### Q：后续会更新吗？

会。彼得学长会根据考试趋势和批改经验持续迭代标准，更新后会发布新版本的 zip 包。建议关注「清新研习社」公众号或 Watch 本仓库获取更新通知。
